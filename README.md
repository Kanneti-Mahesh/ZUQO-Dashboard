# ZUQO-Dashboard

How to run frontend & backend web application dashboard
--------------------------------------------------------

#Client-side:

1. Download and Unzip the files (or) clone github code with command "git clone URL".
2. Open the extracted files with code editor IDE recommended VSCode IDE.
3. Goto terminal of VSCode and change dir to client folder path.
4. Now, type command "npm i" or "npm install" to install necessary packages.
5. After installed packages, finally type "npm start" command to start react-app.



#Server-side:

1. Open new terminal on vscode editor and change dir to server folder path.
2. Now, type command "npm i" or "npm install" to install server-side necessary packages.
3. After installed packages, finally type "npm run server" command to start the server and wait for DB to connect.
